# SnakeGame
Try to increase the length of the snake 🐍

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install packages needed (consolet) 🎁

```bash
pip install -r requirements.txt
```

## Run

```bash
python3 main.py # Mac/Linux
py main.py # Windows
```
